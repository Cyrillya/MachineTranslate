## ✨ 功能
1. 翻译物品简介
2. 自动翻译聊天栏信息
3. 可选多种翻译API
    - 自带无需密钥的免费翻译API
4. 一键打开翻译网站
    - 可选使用Steam自带浏览器打开网站，方便快捷
5. 自动检测并翻译鼠标位置的文本
6. 附带完整的翻译API申请教程
7. 翻译文本缓存，重启游戏后不需要重新翻译
